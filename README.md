# Webpack-Client

### :pushpin: About project
<p align="justify">In this project, I studied the capabilities of webpack and interacting with it with different libraries, I also implemented a panel in the console from the webpack for convenient tracking of information with files.</p>

---

### :book: Libraries

- babel-loader
- cross-env
- css-loader
- css-minimizer-webpack-plugin
- file-loader
- image-webpack-loader
- mini-css-extract-plugin
- sass
- sass-loader
- style-loader
- uglifyjs-webpack-plugin
- webpack
- webpack-cli
- webpack-dashboard
- webpack-dev-server

---

### :pizza: Commands

- npm run dev: <strong>run in development mode</strong>
- npm run build: <strong>assemble the project</strong>
- npm run production:<strong>run the built project</strong>
